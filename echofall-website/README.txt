ECHOFALL - HTML Website
========================

This is your Echofall book converted to an HTML website.

HOW TO VIEW:
-----------
Simply open "index.html" in any web browser to start reading.

STRUCTURE:
----------
- index.html - The home page with table of contents
- prologue.html - The prologue
- the-chronicles-of-evolution.html - The Chronicles of Evolution
- chapter-1-a-frozen-generation.html through chapter-44-option-3.html - All 44 chapters
- styles.css - The stylesheet for the website
- images/ - Folder containing all images from your book

NAVIGATION:
-----------
- Click on any chapter title on the home page to read that chapter
- Each chapter page has "Previous", "Home", and "Next" buttons for easy navigation
- The "Home" button takes you back to the table of contents

FEATURES:
---------
- Beautiful gradient design with purple theme
- Responsive layout that works on desktop, tablet, and mobile
- Easy navigation between chapters
- All images from your Notion export are included
- Clean, readable typography optimized for long-form reading

SHARING:
--------
You can share this website by:
1. Uploading the entire "echofall-website" folder to a web hosting service
2. Sharing the folder with others who can open index.html locally
3. Using GitHub Pages or similar free hosting services

Enjoy your book!
